## 体育比赛直播平台
基于 vue 开发的直播项目，如果觉得项目不错，请帮忙 `Star` 支持一下。

### 技术栈
* vue
* vuex
* vue-router
* element-ui
* ckplayer

### 运行项目
``` bash
# clone 项目
git clone git@github.com:caozongpeng/live-project.git

# 进入项目
cd live-project

# 安装依赖
npm install

# 启动服务
npm run dev

# 打包项目(需要在index.html里引入文件build/build.js)
npm run build

# 打包成功会出现一个 dist 目录，将目录打包上传到服务器用 nginx 部署即可
```
